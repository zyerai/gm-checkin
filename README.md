# 🌅 GM打卡日志

一个简单的每日打卡工具，帮助你养成每天说"GM"的习惯！追踪你的连续打卡天数，生成GM文案，记录每日心情。

## ✨ 功能特点

- ✅ **每日打卡**: 记录每天的GM时刻
- 🔥 **连续打卡**: 自动计算连续打卡天数
- 📝 **心情记录**: 6种心情状态可选
- 📅 **日历视图**: 直观的月度日历展示
- 📊 **数据统计**: 总打卡天数、最长连续、打卡率等
- 🎲 **GM文案生成**: 随机生成GM文案，一键复制
- 💾 **本地存储**: 所有数据存储在本地SQLite数据库
- 🎨 **美观界面**: 渐变色设计，简洁易用

## 📋 快速开始

### 1. 安装依赖

```bash
cd gm-checkin
pip install -r requirements.txt
```

### 2. 运行应用

```bash
python app.py
```

### 3. 访问应用

打开浏览器访问: **http://localhost:5001**

## 📖 使用指南

### 每日打卡

1. 打开应用，点击"立即打卡"按钮
2. 选择今日心情：
   - 🌅 GM - 普通早安
   - 🚀 Bullish - 看涨兴奋
   - 💪 Focused - 专注状态
   - 📚 Learning - 学习模式
   - 😌 Chill - 放松心情
   - ⚡ Grinding - 奋斗中
3. （可选）添加今日备注或计划
4. 点击"打卡！"完成

### 生成GM文案

点击"生成GM文案"按钮，系统会随机生成一条GM文案，包含：
- 日期
- 心情emoji
- 鼓励语

点击"复制"即可复制到剪贴板，方便发到社交媒体！

### 查看历史

点击顶部"历史"标签查看所有打卡记录，可以：
- 查看具体日期的心情和备注
- 删除错误的打卡记录

### 统计分析

点击"统计"标签查看：
- 总打卡天数
- 当前连续打卡天数
- 最长连续打卡天数
- 近30天打卡率
- 本月打卡天数
- 心情分布统计

## 💾 数据存储

所有数据存储在本地的 `gm_tracker.db` SQLite数据库文件中。

**重要提示**:
- 定期备份数据库文件
- 数据库文件位于项目根目录
- 如需迁移到其他电脑，直接复制 `.db` 文件即可

## 🔧 技术栈

- **后端**: Flask (Python)
- **数据库**: SQLite
- **前端**: HTML + CSS + JavaScript
- **无需API**: 所有数据本地存储

## 💡 使用技巧

1. **固定时间打卡**: 每天同一时间打卡，更容易养成习惯
2. **记录计划**: 在备注中写下今天的计划或目标
3. **回顾历史**: 定期查看历史记录，回顾成长
4. **生成文案**: 打卡后生成GM文案发到X/推特
5. **保持连续**: 尽量保持连续打卡，建立成就感
